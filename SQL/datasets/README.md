# Datasets

Keep datasets here or keep only links if the files are too large.

Recommended layout:
- `sql/datasets/raw/`
- `sql/datasets/processed/`
- `sql/datasets/links.md`

Suggested datasets for SQL practice:
- Employees and departments
- Orders and customers
- Online retail
- Sales transactions
- Movie ratings
- Spotify or streaming data exports

If you do not want to commit large files:
- store only small CSV samples here
- put the download link in `links.md`
- mention the dataset in `sql/WORK_LOG_SQL.md`

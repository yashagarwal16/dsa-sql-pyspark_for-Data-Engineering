## How To Use

When you finish a SQL topic:
1. Add the notebook or query file in the correct folder.
2. Replace `To add` with the real file name.
3. Update the best question you solved.
4. Mention the dataset.
5. Mention the path where the file lives.
6. Change status to `Done`.
